# gw2-java

![image](https://github.com/Muqiuer/gw2-java/assets/95432047/33458bbe-bc45-4c1d-9c43-da415f2464ab)<br>

![image](https://github.com/Muqiuer/gw2-java/assets/95432047/9c1d3e59-ddb9-43e2-a49d-ec86c89ce235)<br>
